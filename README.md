## 🎮 Gameplay

The game revolves around a **Shoot → Teleport** traversal mechanic.

1. **Shoot Bubble** – Fire a bubble toward the mouse cursor.
2. **Teleport** – Press again while the bubble is in flight to teleport to it.

Players must retrieve a **key** and unlock the **exit door** while avoiding hazards.
